## Features
- Classic Snake game (Nokia style)
- Responsive & mobile-friendly
- Keyboard and swipe controls
- Pause/Resume support
- 3 difficulty levels
- Score counter
- Modern React + Vite + TypeScript + SCSS Modules

## Getting Started

```bash
npm install
npm run dev
```

## Build & Deploy
